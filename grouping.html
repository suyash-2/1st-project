P element

P element define the paragraph.


Hr element

The hr tag defines a thematic break in an HTML page (e.g. a shift of topic).The hr element is used to separate content (or define a change) in an HTML page.

Pre element

The pre tag defines preformatted text.For example :

	Text in a pre element
	is displayed in a fixed-width
	font, and it preserves
	both spaces and
	line breaks

Blockquote element

The blockquote tag specifies a section that is quoted from another source.Browsers usually indent blockquote elements.

List

There are 3 types of list element they are:

  1. Data element(dl)
  2. Order element(ol)
  3. unorder element(ul)

Dl element

The dl tag defines a description list. The dl tag is used in conjunction with dt (defines terms/names) and dd (describes each term/name). A description list, with terms and descriptions:
Coffee
Black hot drink
Milk
White cold drink

Ol element

The ol tag defines an ordered list. An ordered list can be numerical or alphabetical. Use the li tag to define list items. 2 different ordered lists:
  1. Coffee
  2. Tea
  3. Milk

It can also be started from any random numbers.Like:

  1. Coffee
  2. Tea
  3. Milk

Ul element

The ul tag defines an unorder list.For example:

  • Coffee
  • Tea
  • Milk
